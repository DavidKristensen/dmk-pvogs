Prokaryotic Virus Orthologous Groups

News and Updates - Database History



May 2017 - Viral Quotient (VQ) calculations were made available. VQ values are shown at the top right of each individual VOG table (for instance, VOG0001) and as the second column in the VOG table. Details about how VQ values were calculated is provided at the Tutorial page and here.

May 2017 - Web update to javascript interface. Tables displayed in the website are now sortable and searchable.

January 2017 - pVOGs Database paper published in Nucleic Acids Research Database Issue. Read it here! ➚

October 2016 - First pVOGs Database version was launched.

May 2016 - Collection of prokaryotic virus genomes downloaded from NCBI Nucleotide and analysed for VOG's construction.
