Prokaryotic Virus Orthologous Groups

Download tabular delimited files

1. Genome table: Description of all virus genomes in pVOGs database.

2. VOG table: Description of all virus orthologous groups (VOGs) and their mapping to POGs 2013.

3. VOG Protein table: Description of all proteins found in virus orthologous groups from all virus genomes in pVOGs database (warning - large file - you may want to right-click and choose "Save As").

4. Viral Quotient table: A table with viral quotient measurements for all VOGs.
